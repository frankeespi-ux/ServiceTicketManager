# Service Ticket Manager

A simple Java console application designed to help manage service tickets for technical support scenarios. This project was built as my first real Java application while learning software development fundamentals and object-oriented programming.

## 🚀 Features

- Create new service tickets  
- List all tickets  
- Filter tickets by status (OPEN, IN_PROGRESS, CLOSED)  
- Update the status of existing tickets  
- Automatically assigns IDs and timestamps  
- Clean, object-oriented structure with separate classes:
  - `ServiceTicketManager`
  - `Ticket`
  - `TicketRepository`
  - `TicketStatus` (enum)
  - `TicketPriority` (enum)

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
  - Classes
  - Objects
  - Methods
  - Encapsulation
  - Enums
- Collections (ArrayList)
- Input handling using `Scanner`
- Control flow (`switch`, loops)
- Modular code organization
- Clean separation of concerns
